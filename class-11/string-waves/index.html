Waves on a Stretched String

One-dimensional mechanical wave propagation.

1. Mechanical Waves

A mechanical wave is a disturbance that propagates through a material medium by transfer of energy, not matter.

Particles oscillate locally; the wave travels globally.

2. Transverse Waves on a String

Waves on a stretched string are transverse: particle displacement is perpendicular to direction of propagation.

3. Wave Speed on a String

\[ v = \sqrt{\frac{T}{\mu}} \]
  • \(T\): tension in the string
  • \(\mu\): mass per unit length
Wave speed depends on medium properties, not on how the wave is produced.

4. Wave Equation

\[ y(x,t) = A\sin(kx - \omega t) \]
  • \(k = \frac{2\pi}{\lambda}\)
  • \(\omega = 2\pi f\)
  • \(v = \frac{\omega}{k}\)

5. Energy in a String Wave

Energy propagates along the string through oscillatory motion of particles.

Higher amplitude → higher energy transport.

6. Reflection of Waves

  • Fixed end → phase reversal
  • Free end → no phase reversal

7. Standing Waves on a String

\[ \lambda_n = \frac{2L}{n} \quad (n = 1,2,3,\dots) \]
\[ f_n = \frac{nv}{2L} \]
  • Nodes → zero displacement
  • Antinodes → maximum displacement
Standing waves arise from superposition of identical waves.

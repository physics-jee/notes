Thermodynamics

General laws governing heat, work, and energy.

1. Thermodynamic System

A thermodynamic system is a macroscopic part of the universe chosen for analysis.

  • Open system — exchanges mass and energy
  • Closed system — exchanges energy only
  • Isolated system — exchanges neither
Thermodynamics ignores microscopic details and focuses on macroscopic state.

2. Thermodynamic State & Variables

The state of a system is defined by state variables such as pressure, volume, and temperature.

State variables
  • P, V, T, U
  • Path independent
Path variables
  • Heat (Q), Work (W)
  • Depend on process
A state variable remembers only where the system is — not how it got there.

3. Thermodynamic Process

A process is a transformation from one equilibrium state to another.

  • Isothermal (T constant)
  • Isobaric (P constant)
  • Isochoric (V constant)
  • Adiabatic (Q = 0)

4. Work in Thermodynamics

Work is done when the boundary of a system moves.

\[ W = \int P\,dV \]
  • Area under P–V curve represents work
  • Depends on path followed
Same initial and final states can involve different work.

5. Internal Energy

Internal energy (U) is the total microscopic energy of a system.

  • State function
  • For ideal gas, depends only on temperature

6. First Law of Thermodynamics

The first law is the law of energy conservation for thermodynamic systems.

\[ \Delta U = Q - W \]
  • Q > 0: heat added to system
  • W > 0: work done by system
The first law does not say which processes are possible — only how energy balances.

7. First Law for Special Processes

Isochoric
  • W = 0
  • Q = ΔU
Isothermal (ideal gas)
  • ΔU = 0
  • Q = W
Adiabatic
  • Q = 0
  • ΔU = −W
Isobaric
  • W = PΔV

8. Heat Capacities of Gases

Heat capacity depends on the constraint under which heating occurs.

\[ C_V = \left(\frac{dQ}{dT}\right)_V, \quad C_P = \left(\frac{dQ}{dT}\right)_P \]
Mayer’s relation: \[ C_P - C_V = R \]
Constraints change energy pathways.

9. Cyclic Process

In a cyclic process, the system returns to its initial state.

\[ \Delta U = 0 \Rightarrow Q = W \]
Cycles convert heat into work without storing energy.

10. Common Mistakes

  • Forgetting sign convention in first law
  • Confusing heat with internal energy
  • Assuming work depends only on states
